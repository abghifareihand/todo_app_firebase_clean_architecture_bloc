## :tada: Features App
- Clean Code with Clean Architecture BLOC
- Add Todo with Reminder Notification
- Update Todo
- Update Status Todo (Pending to Completed)
- Get Todo by Date
- Get Pending Todo
- Get Completed Todo
- Authentication Login Register Logout
